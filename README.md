## ⚙️ Installation
- ขั้นตอนโดยละเอียด:โคลนโปรเจกต์ (Clone Project): เปิด Terminal/Git Bash แล้วใช้คำสั่ง:git clone <URL_GitHub_Repository>
- เข้าโฟลเดอร์โปรเจกต์ (Enter Directory):cd <ชื่อโฟลเดอร์โปรเจกต์>
- ติดตั้ง Dependencies (Install Dependencies):composer install (สำหรับ PHP Packages)
- ตั้งค่าสภาพแวดล้อม (Configure Environment):คัดลอกไฟล์ .env.example เป็น .envLinux/macOS: cp .env.example .envWindows: copy .env.example .envสร้าง App Key: php artisan key:generateตั้งค่าฐานข้อมูลในไฟล์ .env (DB_DATABASE, DB_USERNAME, DB_PASSWORD)
- รัน Database Migration (Run Migration):php artisan migrate
- เปิดโปรเจกต์ (Run Application):php artisan serve
